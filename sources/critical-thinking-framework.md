## Overview

A structured approach to critical thinking that supports academic analysis and argument evaluation.

## Framework Components

### Analysis
- Identify the main claim or thesis
- Break down supporting arguments
- Examine evidence quality and relevance
- Identify assumptions (stated and unstated)

### Evaluation
- Assess argument validity and soundness
- Check for logical fallacies
- Evaluate source credibility
- Consider alternative interpretations

### Synthesis
- Connect ideas across sources
- Identify patterns and themes
- Build original arguments from evidence
- Draw supported conclusions

## Application to Essay Writing

Use this framework when:
- Evaluating sources for inclusion in essays
- Structuring arguments logically
- Identifying weaknesses in your own reasoning
- Responding to counter-arguments
